# Mridul Hasan - Portfolio

A modern, responsive portfolio website featuring smooth animations, a typing effect, and clean design. Built to showcase my projects and skills with an elegant user experience.

🌐 **Live Demo:** [View Portfolio](https://mridul.sopdad.com)

## ✨ Features

- ⚡ **Fully Responsive** - Perfect on all devices
- 🎨 **Typing Effect** - Animated text showcasing skills
- 🌍 **Smooth Scroll** - Fluid navigation between sections
- 💻 **Clean Code** - Modular HTML, CSS, and JavaScript
- 📧 **Contact Form** - Integrated with EmailJS for instant messaging

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Animations:** CSS Keyframes, Intersection Observer API
- **Form Handling:** EmailJS
- **Deployment:** GitHub Pages
